این پروژه برای حل مشکل رایج دسترسی به لینک‌های اشتراک در محیط‌هایی طراحی شده است که ممکن است لینک‌ها به‌طور مداوم فیلتر یا مسدود شوند. با استفاده از Cloudflare Workers، یک اسکریپت نوشته‌ام که به‌طور داینامیک لینک‌های اشتراک را با استفاده از دامنه‌های رایگان میزبانی‌شده در Cloudflare ایجاد می‌کند.

ویژگی‌های اصلی:

تعویض داینامیک دامنه‌ها: هر بار که یک لینک مسدود یا فیلتر می‌شود، یک دامنه جدید توسط worker ایجاد می‌شود تا دسترسی به لینک اشتراک بدون وقفه ادامه یابد.

ادغام با Cloudflare Worker: از پلتفرم سرورلس قدرتمند Cloudflare برای ایجاد و مدیریت لینک‌های اشتراک و هدایت آن‌ها به‌صورت بدون مشکل استفاده می‌شود.

میزبانی دامنه‌های رایگان: استفاده از دامنه‌های رایگان ارائه‌شده توسط Cloudflare Workers به این معنی است که نیازی به هاستینگ خارجی یا مدیریت دامنه اضافی نیست.

نحوه عملکرد:

اسکریپت در محیط Cloudflare Worker راه‌اندازی می‌شود.

لینک اشتراک جدیدی با استفاده از داده‌های JSON ایجاد می‌شود.

در صورت مسدود شدن یا فیلتر شدن، worker به‌طور خودکار به یک دامنه جدید رایگان از Cloudflare سوئیچ می‌کند.

این راه‌حل اطمینان می‌دهد که لینک‌های اشتراک علی‌رغم مسدود شدن یا فیلتر شدن دامنه‌های قبلی، همچنان به‌طور پایدار کار می‌کنند.
